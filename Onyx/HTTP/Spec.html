    module Onyx::HTTP::Spec

    Overview

    HTTP testing module. It helps to test your Onyx application. require "onyx/http/spec" before your application, so it spawn a server instead of blocking on Onyx::HTTP.listen.

    Defined in:

    onyx/http/spec.cr
